# 🎓 Student Management System

A Java console-based application for managing student records efficiently. This project demonstrates Object-Oriented Programming (OOP) principles, exception handling, input validation, and modular code organization.

## 📌 Features

- ➕ Add Student
- 📋 Display All Students
- 🔍 Search Student by ID
- ✏️ Update Student Details
- ❌ Delete Student
- 🆔 Automatic Student ID Generation
- ✅ Input Validation
- ⚠️ Custom Exception Handling
- 🗂️ Modular Project Structure

---

## 🛠️ Technologies Used

- Object-Oriented Programming (OOP)
